## Overview

This file defines positive-definite functions on additive groups and proves their fundamental properties. A function $\varphi : \alpha \to \mathbb{C}$ is positive definite if it satisfies Hermitian symmetry ($\varphi(-x) = \overline{\varphi(x)}$) and the matrix condition that $\sum_{i,j} \overline{c_i} c_j \varphi(x_i - x_j) \geq 0$ for all finite point sets and coefficient vectors. The Schur product (pointwise multiplication) of two positive-definite functions is positive definite, a key fact for Gaussian regularization in Bochner's theorem.

## Structure `IsPositiveDefinite`
**Location**: [Line 63](../../Bochner/PositiveDefinite.lean#L63)

A function $\varphi : \alpha \to \mathbb{C}$ is positive definite if:
- `hermitian`: $\varphi(-x) = \overline{\varphi(x)}$ for all $x$ (Hermitian symmetry ensures the kernel matrix is Hermitian)
- `nonneg`: $\sum_{i,j} \overline{c_i} c_j \varphi(x_i - x_j) \geq 0$ in the real part (positive semidefiniteness)

Together these encode that the matrix $[\varphi(x_i - x_j)]$ is Hermitian positive semidefinite.
