# Sazonov's Theorem — Formalization Plan

## Motivation

Sazonov's theorem is the mathematically correct way to bypass the paradoxical
`nuclear_support_concentration` axiom. It is the "final boss" of the
Gel'fand-Vilenkin approach to infinite-dimensional probability.

A nuclear space is defined by a family of seminorms where the canonical
inclusions between their local Hilbert spaces are Hilbert-Schmidt operators.
The topology of a nuclear space is essentially a Sazonov topology. Proving
Sazonov's theorem on a Hilbert space collapses Axiom 1
(`nuclear_support_concentration`) into a trivial corollary.

## 1. Feasibility Assessment

### What Mathlib 4 already provides

- **Hilbert Spaces:** `InnerProductSpace ℝ H` and `CompleteSpace H` are fully
  developed.
- **Prokhorov's Theorem:** `Mathlib.MeasureTheory.Measure.Prokhorov` contains
  the proof that uniformly tight families of measures have weakly convergent
  subsequences (`isCompact_closure_of_isTightMeasureSet`).
- **Finite-Dimensional Integration:** Fubini's theorem and finite-dimensional
  Gaussian measures (`MeasureTheory.Measure.gaussian`).

### What Mathlib 4 does NOT have

- **Infinite-Dimensional Trace-Class Operators:** Mathlib has finite-dimensional
  matrix traces and is actively merging PRs on operator algebras, but a robust
  API for infinite-dimensional trace-class and Hilbert-Schmidt operators is
  still missing.
- **The Sazonov Topology:** Must be defined manually as a locally convex
  topology generated by seminorms of the form x ↦ √⟪Sx, x⟫ for positive
  trace-class S.
- **The Sazonov Tightness Bound:** The core Chebyshev-Markov inequality linking
  the characteristic functional to the tails of the measure.

## 2. Strategy: The Fubini/Gaussian Trick

Standard textbooks prove Sazonov via cylinder measures and finitely-additive
integrals. **Do not do this in Lean.** Formalizing finitely-additive integrals
over algebras of cylinder sets is notoriously painful.

Instead, since we already built `marginalProjectiveLimit`, prove Sazonov
entirely via **uniform tightness of finite-dimensional marginals**:

1. **Finite-Dimensional Bound:** Let μ_F be the marginal measure on a
   finite-dimensional subspace F, and φ its characteristic function. If φ is
   continuous in the Sazonov topology, it is bounded by a trace-class
   operator S.

2. **Fubini's Theorem:** Integrate 1 - Re(φ(x)) against a finite-dimensional
   Gaussian with covariance S. Swapping integrals via Fubini gives
   ∫_F (1 - exp(-½⟪Sy, y⟫)) dμ_F(y).

3. **Tightness:** By Markov's inequality, this bounds the mass of μ_F outside
   the ellipsoid {y : ⟪Sy, y⟫ > R²}. Because S has finite trace, this bound
   is **independent of dim(F)**.

4. **Prokhorov:** Uniform tightness across all finite-dimensional subspaces →
   feed into Mathlib's Prokhorov theorem → countably additive Radon measure
   on the Hilbert space H.

## 3. Lean 4 Blueprint

### Phase 1: Trace-Class Operators (Algebraic Gap)

```lean
import Mathlib.Analysis.InnerProductSpace.Basic
import Mathlib.MeasureTheory.Measure.ProbabilityMeasure
import Mathlib.MeasureTheory.Measure.Prokhorov
import Mathlib.Analysis.LocallyConvex.WithSeminorms

open MeasureTheory Complex Filter Topology Set
open scoped Real InnerProductSpace

variable {H : Type*} [NormedAddCommGroup H] [InnerProductSpace ℝ H] [CompleteSpace H]

/-- A continuous linear operator is positive if ⟪Sx, x⟫ ≥ 0 for all x. -/
def IsPositiveOperator (S : H →L[ℝ] H) : Prop :=
  ∀ x : H, 0 ≤ @inner ℝ H _ x (S x)

/-- Trace-class operators: the trace over an orthonormal basis is finite
    and basis-independent. -/
def IsTraceClass (S : H →L[ℝ] H) : Prop := sorry

structure PositiveTraceClassOperator (H : Type*) [NormedAddCommGroup H]
    [InnerProductSpace ℝ H] [CompleteSpace H] where
  op : H →L[ℝ] H
  is_pos : IsPositiveOperator op
  is_trace_class : IsTraceClass op
```

### Phase 2: The Sazonov Topology (Topological Gap)

```lean
/-- Every positive trace class operator S defines a seminorm x ↦ √⟪Sx, x⟫. -/
noncomputable def sazonovSeminorm (S : PositiveTraceClassOperator H) : Seminorm ℝ H where
  toFun x := Real.sqrt (@inner ℝ H _ x (S.op x))
  map_zero' := sorry
  add_le' := sorry
  neg' := sorry
  smul' := sorry

/-- The Sazonov topology is generated by the family of all Sazonov seminorms.
    Use Mathlib's `WithSeminorms` for locally convex topology generation. -/
def sazonovTopology (H : Type*) [NormedAddCommGroup H] [InnerProductSpace ℝ H]
    [CompleteSpace H] : TopologicalSpace H :=
  TopologicalSpace.generateFrom { U | sorry }
```

### Phase 3: Sazonov Tightness Lemma (Analytic Core)

```lean
def IsPositiveDefinite (φ : H → ℂ) : Prop := sorry

/-- The characteristic functional of a measure on a Hilbert space. -/
noncomputable def charFun (μ : Measure H) (x : H) : ℂ :=
  ∫ y, cexp (I * ↑(@inner ℝ H _ x y)) ∂μ

/-- The Sazonov Tightness Lemma (Fubini/Gaussian Bound).
    If φ is continuous at 0 in the Sazonov topology, the finite-dimensional
    marginals are uniformly tight. -/
lemma sazonov_tightness (φ : H → ℂ) (hpd : IsPositiveDefinite φ) (hnorm : φ 0 = 1)
    (hsaz : @ContinuousAt H ℂ (sazonovTopology H) _ φ 0) :
    True := by sorry
```

### Phase 4: Sazonov's Theorem

```lean

    unique Radon probability measure. -/
theorem sazonov_theorem (φ : H → ℂ)
    [MeasurableSpace H] [BorelSpace H]
    (hpd : IsPositiveDefinite φ)
    (hnorm : φ 0 = 1)
    (hsaz : @Continuous H ℂ (sazonovTopology H) _ φ) :
    ∃! μ : ProbabilityMeasure H, ∀ x : H, charFun μ.toMeasure x = φ x := by
```

## 4. Minlos via Sazonov

Once `sazonov_theorem` is proved on a Hilbert space, Minlos' theorem follows
cleanly:

1. The continuous functional on the nuclear space E factors through the
   completion of one of its continuous seminorms, giving a Hilbert space H_p.
2. Because the transition maps in a nuclear space are Hilbert-Schmidt,
   Sazonov's theorem triggers on H_p, yielding a Radon measure on H_p'.
3. Push this measure forward to E' via `Measure.map`.

No algebraic duals. No `MeasurableEmbedding` paradoxes. Clean formal
mathematics.

```
  ├── sazonovSeminorm (depends on Phase 1)

  └── sazonov_tightness (depends on Phase 2 + Bochner marginals)
```

## 6. Parallelization

Phases 1-2 (operator algebra + topology) can be developed independently from
Phase 3 (tightness estimates). The Bochner marginal construction is already
complete and can be reused directly.
