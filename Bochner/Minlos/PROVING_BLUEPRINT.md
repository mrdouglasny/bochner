# Blueprint: Proving the 6 Axioms + 1 Sorry (Road to 0)

## Current State

**Files**: `MeasurableModification.lean` (6 axioms), `Minlos.lean` (1 sorry)

**Architecture** (now mathematically sound):
- `qLinearPaths` / `boundedPaths` use `ℕ →₀ ℚ` (Finsupp) for full ℚ-span coverage
- Axioms take `h_cf_joint` (joint finite-dimensional CFs), not tautological 1D CFs
- `h_cf_joint` derives 1D CFs via `Fin 1` specialization (already proved in `goodPaths_ae`)

## Recommended Proving Order

### 1. `h_cf_joint` (sorry in Minlos.lean) — Medium

**What**: Joint characteristic function from projective limit.
```
∀ (n : ℕ) (s : Fin n → ℝ) (x : Fin n → E),
  ∫ ω, exp(I * ↑(∑ i, s i * ω(x i))) ∂ν = Φ(∑ i, s i • x i)
```

**Strategy**: Generalize the existing `h_cf_eq` proof (Minlos.lean lines 82-117)
from singleton `{f}` to arbitrary `Finset.image x Finset.univ`.
Same structure: factor through `Finset.restrict`, use `hν_proj`, change variables
through `finsetPiMeasEquiv`, connect inner product to marginalCF.

**Complication**: When `x : Fin n → E` has duplicates, `Finset.image x Finset.univ`
has fewer elements than `n`. Need to group coefficients by test vector.

**Mathlib tools**: Same as existing `h_cf_eq` proof + `Finset.sum_fiberwise`.

### 2. `qLinearPaths_ae` (axiom 4) — Medium

**What**: ω is ℚ-linear on range(d) for ν-a.e. ω.

**Strategy**:
1. Fix `c : ℕ →₀ ℚ`. Let `y = c.sum (fun i a => (a : ℝ) • d i)`.
2. Define `X(ω) = ω(y) - c.sum (fun i a => (a : ℝ) * ω(d i))`.
3. Show `CF(X)(t) = 1` for all `t`:
   - Construct `Fin (|c.support| + 1)` array with x₀ = y, xₖ₊₁ = d(iₖ).
   - Use `h_cf_joint` with coefficients (t, -tc₁, ..., -tcₖ).
   - Sum of coefficients • vectors = t•y - t•∑cᵢ•dᵢ = 0, so CF = Φ(0) = 1.
4. CF ≡ 1 implies X = 0 a.s.: push forward ν through X, apply
   `Measure.ext_of_charFun` against Dirac δ₀.
5. Countable intersection over `c : ℕ →₀ ℚ` via `ae_all_iff`
   (= `eventually_countable_forall`).

**Mathlib tools**:
- `ae_all_iff` (`Mathlib.MeasureTheory.OuterMeasure.AE`)
- `Measure.ext_of_charFun` (already used in project)
- `MeasureTheory.charFun` for Dirac measure

### 3. `measurable_measurableProjection` (axiom 3) — Medium

**What**: P : (E → ℝ) → WeakDual ℝ E is measurable.

**Strategy**:
1. WeakDual Borel σ-algebra is generated by evaluations (weak-* topology).
   Suffices to show `fun ω => P(ω)(f)` is measurable for each `f : E`.
2. Fix `f : E`. Choose sequence `xₙ → f` from ℚ-span of range(d).
3. On `goodPaths` (measurable): `P(ω)(f) = lim ω(xₙ)` (extension is continuous).
   Each `ω ↦ ω(xₙ)` is `measurable_pi_apply`. Pointwise limit is measurable
   by `measurable_of_tendsto_metrizable`.
4. On `goodPathsᶜ`: `P(ω)(f) = 0` (constant, measurable).
5. Use `Measurable.piecewise` to combine.

**Mathlib tools**:
- `Measurable.piecewise` (`Mathlib.MeasureTheory.MeasurableSpace.Basic`)
- `measurable_of_tendsto_metrizable` (`Mathlib.MeasureTheory.Constructions.BorelSpace.Metrizable`)
- `measurable_pi_apply`

---

### 4. `extensionCLM` & `extensionCLM_eq_on_dense` (axioms 1 & 2) — Medium

**What**: Construct ContinuousLinearMap from ℚ-linear bounded function on dense set.

**Strategy**:
1. On `goodPaths`, `ω` restricted to range(d) satisfies:
   `|ω(dᵢ) - ω(dⱼ)| ≤ C * q(dᵢ - dⱼ)` (from ℚ-linearity + boundedness).
   This makes ω uniformly continuous on range(d) w.r.t. seminorm q.
2. Use `Dense.extend` (`Mathlib.Topology.DenseEmbedding`) to get continuous
   `g : E → ℝ` with `g(dₙ) = ω(dₙ)`.
3. Prove `g` is ℚ-linear: `g(a•x + b•y) = a•g(x) + b•g(y)` for `a, b ∈ ℚ`.
   Both sides are continuous and agree on range(d)×range(d) (dense).
   Use `Continuous.ext_on` (already used in `extensionCLM_embed`).
4. Promote to ℝ-linearity: `g(r•x) = r•g(x)` for `r ∈ ℝ`.
   Both sides continuous in r, agree for r ∈ ℚ (dense in ℝ).
5. Package as `ContinuousLinearMap`.

**Mathlib tools**:
- `Dense.extend` / `IsDenseInducing.extend` (`Mathlib.Topology.DenseEmbedding`)
- `Dense.extend_eq` (for axiom 2: extension agrees on dense set)
- `Continuous.ext_on` (for uniqueness arguments)

---

### 5. `projection_ae_eq` (axiom 6) — Medium

**What**: P(ω)(f) = ω(f) for ν-a.e. ω.

**Strategy**:
1. Fix `f`. Take `xₙ → f` from dense set.
2. On goodPaths: `P(ω)(xₙ) = ω(xₙ)` (by extensionCLM_eq_on_dense) and
   `P(ω)(xₙ) → P(ω)(f)` (P(ω) is continuous).
3. Show `ω(xₙ) → ω(f)` in measure:
   - Let `Yₙ(ω) = ω(f) - ω(xₙ)`.
   - CF of Yₙ at t: use `h_cf_joint` with (f, xₙ), coefficients (t, -t).
     CF = Φ(t(f - xₙ)) → Φ(0) = 1 by continuity.
   - CF → 1 implies convergence in distribution to δ₀, hence in measure.
4. Convergence in measure → subsequence converges a.e.
   (`TendstoInMeasure.exists_seq_tendsto_ae`).
5. So `ω(xₙₖ) → ω(f)` a.e. But also `ω(xₙₖ) = P(ω)(xₙₖ) → P(ω)(f)` a.e.
   Hence `P(ω)(f) = ω(f)` a.e.

**Mathlib tools**:
- `TendstoInMeasure.exists_seq_tendsto_ae`
  (`Mathlib.MeasureTheory.Function.ConvergenceInMeasure`)
- `Measure.ext_of_charFun` (to show CF → 1 implies δ₀)
- Markov inequality for CF convergence → convergence in measure

---

### 6. `boundedPaths_ae` (axiom 5) — Hard

**What**: ∃ s, C such that |ω(x)| ≤ C * (s.sup p)(x) for all x in ℚ-span, a.e.

**Strategy**:
1. For each `x` in the ℚ-span, Markov/Chebyshev gives:
   `P(|ω(x)| ≥ R) ≤ (1 - Re(Φ(tx))) / (1 - cos(tR))`
2. By continuity of Φ at 0: for any ε > 0, ∃ δ > 0 s.t.
   `|1 - Re(Φ(y))| < ε` whenever `p_n(y) < δ` (for some seminorm p_n).
3. For x with `p_n(x) < δ/t`: the numerator < ε, so `P(|ω(x)| ≥ R) < ε/(1-cos(tR))`.
4. This is connected to SazonovTightness.lean which already proves these
   bounds for finite-dimensional marginals.
5. Let `Aₙ = {bounded on first N Finsupp combinations}`. These are decreasing.
   By `Antitone.measure_iInter`, `ν(⋂ Aₙ) = lim ν(Aₙ)`.
   Each `ν(Aₙ) ≥ 1 - εₙ` from the finite-dimensional bounds.

**Mathlib tools**:
- `Antitone.measure_iInter` (`Mathlib.MeasureTheory.Measure.MeasureSpace`)
- SazonovTightness.lean infrastructure (Chebyshev bounds, Gaussian averaging)
- `NuclearSpace.nuclear_hilbert_embeddings` for the seminorm control
